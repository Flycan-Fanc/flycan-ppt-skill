# 黑板手绘板书风 · 知识关系构图库

严格参考 `黑板手绘版书风.png` 和当前 Canvas 原型。本风格是粉笔知识插画分镜，不是课堂实拍、黑色主题 PPT 或标准组件拼装。

生成前同时阅读 `chalk-rendering-blackboard.md`。

## 核心判断

> 第一眼先看见一个由大字、插画和箭头共同表达的观点，再看见粉笔材质。

- 基本视觉单位是“手绘视觉语句”，不是卡片。
- 每页围绕一个核心命题组织内容。
- 页数由内容决定，不固定为 6 页。
- 图形承担叙事，不做装饰性图标陈列。
- 允许不直、不圆、不对称和轻微不闭合。

## 页面规划

每页先写四项：

1. **命题**：本页只回答什么问题？
2. **主插画**：哪个对象能一眼表达命题？
3. **知识关系**：因果、流程、对比、辨析还是结论？
4. **颜色语义**：哪些词或路径需要黄、蓝、粉、绿？

然后再确定文字和图形坐标。不要先套分栏、框线或卡片。

## 构图骨架

以下是可组合的知识关系，不是固定页序。

### 1. 命题主插画

- 用大面积手写标题建立第一视觉重心。
- 用灯泡、大脑、人物、靶心等主插画承载主题。
- 用一组云朵或短批注解释“它不是什么”。
- 用箭头把批注带回核心判断。

适合：封面、提出问题、核心观点。

### 2. 因果推导

- 将 2-4 个原因沿阅读方向排布。
- 用弯箭头汇聚到结果或判断。
- 结果可用大字、圈注或不规则云朵表达，不使用 UI 结果卡。

适合：为什么、问题来源、影响分析。

### 3. 连续流程

- 使用 3-6 个可识别的手绘对象。
- 图形下只放短标签，解释文字贴近对应图形。
- 箭头是流程的一部分，不能用机械细线或标准连接器。

适合：步骤、算法、思考路径、方法论。

### 4. 对比辨析

- 使用天平、分叉路径、左右人物或相互指向的圈注。
- 用 `≠`、`→`、手绘下划线和颜色强调差异。
- 对比双方保持共同视觉语法，不做两张并排 UI 卡片。

适合：误区 vs 本质、事实 vs 观点、工具 vs 能力。

### 5. 方法拆解

- 用大脑、放大镜、勾选轨迹、道路或阶梯作为主结构。
- 方法项围绕主结构生长，不做规整 checklist 组件。
- 每项用短语，必要时增加一行手写注释。

适合：行动建议、训练步骤、学习方法。

### 6. 结论收束

- 用一组大面积金句或判断占据主要空间。
- 保留一个能强化结论的图形，如靶心、灯泡、星号或箭头。
- 署名、页码和装饰退居次级。

适合：总结、价值、行动号召。

## Canvas 页面骨架

每页优先使用固定逻辑尺寸 Canvas：

```html
<section class="slide">
  <canvas
    class="chalk-canvas"
    width="1536"
    height="864"
    aria-label="本页内容的完整文字描述"
  ></canvas>
</section>
```

为每页编写独立 `drawPageN(ctx, api)`，并复用模板中的生成式粉笔 API：

```js
function drawPage1(ctx, api) {
  const { C, chalkText, chalkPath, curve, smoothClosed } = api;

  chalkText("本页核心命题", 80, 180, 92, C.white, 101, {
    rotate: .05,
    baseline: 8,
    scale: .1,
    spacing: 4
  });

  // 继续绘制主插画、箭头、圈注和结论。
}
```

要求：

- 每个元素使用稳定 seed。
- 标题、正文和图形使用不同材质参数。
- 不改变路径轮廓来伪造材质。
- 不把 Canvas 拆成现代 UI 小组件。
- Canvas 必须有准确 `aria-label`，必要时补充屏幕阅读器文本。

## 配色

| 颜色 | 用法 |
|---|---|
| 白色 | 主体文字、主轮廓 |
| 黄色 | 核心词、关键判断、重点路径 |
| 蓝色 | 问题、提示、探索 |
| 粉色 | 强调、转折、结论 |
| 绿色 | 分析、判断、正向关系 |

颜色必须承担语义，不按“每页凑齐五色”使用。

## 禁止

- 固定套用“大问题 → 原因 → 流程 → 方法 → 价值 → 总结”的 6 页模板
- 把细白边框、页码和横线当成每页主结构
- 规则分栏、网格卡片、圆角容器、阴影和玻璃拟态
- Lucide、emoji、真实照片、3D 图标、标准图标式 SVG
- 小而规整的网页正文
- 用大面积背景噪声代替文字和图形的粉笔材质
- 只有文字、没有主插画或知识关系
