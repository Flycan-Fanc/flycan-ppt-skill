# Midnight Minimal · 4 套主题色预设

在 HTML 的 `<style>` 区域覆盖 `:root` 中对应的 CSS 变量即可。不需要改 HTML body。

```html
<!-- 示例：在 index.html 的 <style> 块中替换 :root 变量 -->
<style>
  :root{
    --mm-bg:#0A0A0A;
    --mm-accent:#E8E8E8;
    --mm-accent-rgb:232,232,232;
    --mm-text-soft:rgba(240,240,240,.7);
    --mm-text-dim:rgba(240,240,240,.35);
    --mm-surface:rgba(255,255,255,.04);
    --mm-line:rgba(255,255,255,.08);
  }
</style>
```

## 🌌 星尘 · Stardust

经典银白，如午夜星空。

| 变量 | 色值 | 用途 |
|------|------|------|
| `--mm-bg` | `#0A0A0A` | 深空底色 |
| `--mm-accent` | `#E8E8E8` | 银白强调 |
| `--mm-text` | `#F0F0F0` | 主文字 |
| `--mm-text-soft` | `rgba(240,240,240,.7)` | 次要文字 |
| `--mm-text-dim` | `rgba(240,240,240,.35)` | 弱化文字 |
| `--mm-surface` | `rgba(255,255,255,.04)` | 卡片底色 |
| `--mm-line` | `rgba(255,255,255,.08)` | 分割线 |

**适合**：默认通用、长期主义、哲学思考、人生感悟
