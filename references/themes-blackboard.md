# Blackboard Sketch · 主题色

Blackboard Sketch（黑板手绘板书风）使用固定的黑板配色方案。所有颜色在模板中已预设，不需要额外选择。

## 配色方案

| 颜色 | 色值 | 用途 |
|------|------|------|
| 黑板底色 | `#1B2F1D` | 背景（深绿色黑板） |
| 粉笔白 | `#F5F0E8` | 正文、标题、主要文字 |
| 粉笔黄 | `#E8D56E` | 高亮、重点内容、强调 |
| 粉笔蓝 | `#7AADD9` | 副标题、辅助信息、链接 |
| 粉笔红 | `#D9756B` | 强调、错误、标注、圈注 |
| 粉笔灰 | `#9A9A9A` | 次要文字、注释 |
| 粉笔淡 | `rgba(245,240,232,0.6)` | 淡入效果、次要信息 |

## 使用规则

- 正文使用**粉笔白**，确保在黑底上清晰可读
- 重点内容用**粉笔黄**高亮（像老师在黑板上用黄粉笔强调）
- 副标题/分类用**粉笔蓝**（区分层级）
- 关键强调用**粉笔红**（像老师用红粉笔圈注）
- **不要使用纯黑、纯白、高饱和荧光色**

## 字体建议

- 标题：手写感字体（如 `Liu Jian Mao Cao` / `ZCOOL XiaoWei` / `Ma Shan Zheng`），CDN 加载
- 正文：清晰可读的非衬线字体（`Noto Sans SC`）
- 等宽：`JetBrains Mono`（用于代码片段）

## 禁止

- 彩色背景（黑板绿色是唯一背景色）
- 渐变色
- 阴影（粉笔字没有阴影）
- 圆角卡片（不要破坏黑板质感）
- 图片作为背景（黑板就是背景）
- 商业模板风、企业模板风、营销号风
