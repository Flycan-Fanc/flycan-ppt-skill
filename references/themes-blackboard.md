# 黑板手绘板报风 · 配色预设

## 固定配色（模板中已预设）

黑板手绘板报风使用 **纯黑背景 + 彩色粉笔** 体系，配色在模板中已固定预设，不可按主题切换。

| 颜色 | 色值 | 用途 | 粉笔类名 |
|------|------|------|----------|
| 黑板底色 | `#0B0B0B` | 背景 | — |
| 白粉笔 | `#F5F5F5` | 正文/主标题 | —（默认色） |
| 黄粉笔 | `#FFD54F` | 关键词/重点/高亮 | `.chalk-yellow` |
| 蓝粉笔 | `#64B5F6` | 流程/逻辑关系/连接 | `.chalk-blue` |
| 粉粉笔 | `#F48FB1` | 强调/警示/圈注 | `.chalk-pink` |
| 绿粉笔 | `#81C784` | 成长/正向/确认 | `.chalk-green` |
| 橙粉笔 | `#FFB74D` | 行动建议/实践步骤 | `.chalk-orange` |
| 红粉笔 | `#EF5350` | 风险/错误/误区 | `.chalk-red` |
| 灰色 | `#666666` | 辅助/网格/次要 | `.chalk-dim` |

## 使用规则

| 用法 | 颜色 |
|------|------|
| 标题、正文 | 白色 `#F5F5F5` |
| 关键词、核心概念 | 黄色 `#FFD54F` |
| 流程图、连接线、逻辑关系 | 蓝色 `#64B5F6` |
| 强调、警示、重点框 | 粉色 `#F48FB1` |
| 成长、长期主义、正向内容 | 绿色 `#81C784` |
| 行动建议、实践步骤 | 橙色 `#FFB74D` |
| 风险提醒、错误示例 | 红色 `#EF5350` |
| 次要文本、辅助线 | 灰色 `#666666` |

## 字体推荐

| 角色 | 字体 |
|------|------|
| 标题 | ZCOOL XiaoWei / Ma Shan Zheng（手写体） |
| 正文 | Noto Sans SC（非衬线，清晰） |
| 代码/元数据 | JetBrains Mono（等宽） |

## 禁止

- ❌ 渐变背景
- ❌ 毛玻璃 / Glassmorphism
- ❌ 真实照片、3D 图标
- ❌ 商务/企业/科技炫酷风格
- ❌ 极细字体、商务字体
