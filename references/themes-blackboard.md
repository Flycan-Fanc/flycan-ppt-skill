# 黑板手绘板报风 · 配色预设

## 固定配色（模板中已预设）

黑板手绘板报风使用 **深灰绿石板黑板底色 + 彩色粉笔** 体系，配色在模板中已固定预设，不可按主题切换。

| 颜色 | 色值 | 用途 | 类名 |
|------|------|------|------|
| 黑板底色 | `#202321` | 石板黑板背景 | — |
| 白粉笔 | `#F5F5F5` | 默认正文/标题 | —（默认色） |
| 黄粉笔 | `#FFD54F` | 关键词/重点/核心问题 | `.chalk-yellow` |
| 蓝粉笔 | `#64B5F6` | 流程/逻辑/分析推导 | `.chalk-blue` |
| 粉粉笔 | `#F48FB1` | 强调/警示/有趣观点 | `.chalk-pink` |
| 绿粉笔 | `#81C784` | 成长/确认/正向 | `.chalk-green` |
| 橙粉笔 | `#FFB74D` | 行动建议/实践步骤 | `.chalk-orange` |
| 红粉笔 | `#EF5350` | 风险/错误/否定（少用） | `.chalk-red` |
| 灰粉笔 | `rgba(245,245,245,.35)` | 辅助/标签/次要 | `.chalk-dim` |

## 粉笔粗细层级

不同层级使用不同粗细的粉笔（通过字体大小 + text-shadow 模拟）：

| 层级 | 字体 | 粗细感 | 适合 |
|------|------|--------|------|
| `.chalk-title` 6.5vw | ZCOOL XiaoWei | **粗粉笔**（重阴影） | 封面巨标题 |
| `.chalk-h1` 4.8vw | ZCOOL XiaoWei | **中粉笔**（中阴影） | 章节大标题 |
| `.chalk-h2` 3.2vw | Ma Shan Zheng | **细粉笔手写** | 副标题/引导语 |
| `.chalk-body` 1.2vw 300w | Noto Sans SC | **极细粉笔** | 正文内容 |
| `.chalk-note` 1.1vw | Ma Shan Zheng | **粉笔注释** | 旁注/说明 |
| `.chalk-mono` 0.75vw | JetBrains Mono | **刻印/标签** | 类别标识 |

## 使用规则

| 用法 | 颜色 |
|------|------|
| 标题、正文主内容 | 白色 |
| 关键词、核心概念、问题 | 黄色 |
| 流程图、推导过程、逻辑关系 | 蓝色 |
| 强调、警示、有趣的观点 | 粉色 |
| 成长、长期主义、正向内容 | 绿色 |
| 行动建议、实践步骤 | 橙色 |
| 风险提醒、错误示例 | 红色（少用） |
| 辅助文字、页码、标签 | 灰色 |

## 字体推荐

| 角色 | 字体 |
|------|------|
| 巨标题 / 大标题 | ZCOOL XiaoWei（粗重手写，适合厚粉笔） |
| 副标题 / 注释 | Ma Shan Zheng（自然行书，适合细粉笔） |
| 正文结构化内容 | Noto Sans SC 300（超细体，适合极细粉笔） |
| 代码/标签 | JetBrains Mono（等宽） |

## 黑板纹理说明

模板中的黑板纹理由多层 CSS 渐变模拟：

1. **粉笔灰痕迹**：多个 `radial-gradient` 椭圆斑点
2. **板擦纹**：`linear-gradient` 倾斜条纹
3. **木纹**：`repeating-linear-gradient` 水平细线
4. **粉笔字毛糙感**：`text-shadow` 多层偏移 + 模糊

## 禁止

- ❌ 渐变背景（纹理除外）
- ❌ 毛玻璃 / Glassmorphism
- ❌ 真实照片、3D 图标、emoji
- ❌ 商务/企业/科技炫酷风格
- ❌ 极细无衬线字体作标题
- ❌ 杂志式排版（chrome、foot 等）
