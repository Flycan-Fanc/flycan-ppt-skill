# 黑板手绘板书风 · 视觉规范

本风格严格参考根目录图片 `黑板手绘版书风.png` 和当前 Canvas 原型。

核心不是“真实课堂黑板”或“黑色高级 PPT”，而是：

> 由老师用彩色粉笔画出的知识插画分镜。

## 一眼识别标准

- 大面积手写标题与可识别主插画共同构成视觉重心。
- 箭头、流程、圈注和颜色直接表达知识关系。
- 信息密度较高，但阅读顺序由尺度、颜色和空间邻近控制。
- 线条允许不直、不圆、不对称和轻微不闭合。
- 黑板仅是低调画布，粉尘和擦痕不能压过内容。
- 页数由内容决定，不固定为 6 页。

## 固定配色

以当前 Canvas 原型为主：

| 颜色 | 色值 | 语义 |
|---|---|---|
| 黑板底色 | `#101411` | 低调画布 |
| 白粉笔 | `#FFF9E9` | 主体文字、主轮廓 |
| 黄粉笔 | `#FFD52F` | 核心词、关键判断 |
| 蓝粉笔 | `#4DC8FF` | 问题、提示、探索 |
| 粉粉笔 | `#FF6294` | 强调、转折、结论 |
| 绿粉笔 | `#89E052` | 分析、判断、正向关系 |

不要为了丰富而新增橙色、红色或更多主题色。每种颜色必须承担稳定语义。

## 字体与字形

- 使用可用的中文手写字体作为字形骨架：
  `"HanziPen SC","Hannotate SC","Kaiti SC","STKaiti",cursive`
- 标题逐字改变字号、基线、旋转和字距，形成大、粗、略歪的海报字重心。
- 正文仍保留轻微手写变化，但连续覆盖和可读性优先。
- 不依赖 CSS `text-shadow` 或 SVG filter 单独制造粉笔感。
- 使用离屏 Canvas 处理字符 alpha 和粉体覆盖，详见 `chalk-rendering-blackboard.md`。

## 图形语言

- 灯泡、问号、放大镜、天平、靶心、大脑、人物、星星和云朵是叙事词汇，不是必须凑齐的图标列表。
- 主图形应比辅助粉尘更醒目。
- 图形优先由 `chalkPath`、`curve`、`smoothClosed` 等 Canvas 路径直接绘制。
- 箭头和圈注必须连接观点，不能只作装饰。
- 禁止规整图标比例、机械圆形、标准连接器和组件库式重复。

## 黑板背景

- 使用低对比径向擦痕和稀疏粉尘颗粒。
- 细边界、木框和页码可以保留，但不能主导构图。
- 不模拟真实教室光影、反光、摄影景深或厚重木纹。

## 禁止

- Apple 发布会式居中大标题和大面积空黑
- 极简杂志排版、设计作品集排版
- 现代 UI 卡片、规则分栏、圆角阴影、玻璃拟态
- Lucide、emoji、真实照片、3D 图标、标准图标式 SVG
- 小而规整的网页正文
- 用背景颗粒或滤镜替代文字与路径本身的粉笔材质
